## usage
```javascript
npm install -g @stackr23/styleobjects
npx @stackr/styleobjects --input SRC --output DEST
```

__@param input__&nbsp;&nbsp;&nbsp;- {`String:path` | `String` | `URL` | `Buffer`} - __required__  
__@param outout__&nbsp;- {`String:path`}
